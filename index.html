Empat Mod Pembelajaran

1. Latihan Solo

Belajar mengikut rentak sendiri tanpa tekanan masa untuk membina dan memperkukuhkan asas pengiraan.

Pangkat Boleh Dicapai:
NOVIS ➔ PERANTIS ➔ MAHIR ➔ PAKAR ➔ JAGUH

2. Penilaian Standard

Ujian berstruktur untuk menilai tahap penguasaan semasa pelajar secara keseluruhan.

Sistem Penggredan:
GRED D (ASAS) ➔ GRED C ➔ GRED B ➔ GRED A ➔ GRED S (SEMPURNA)

3. Arena Kompetitif

Uji kelajuan dan ketepatan anda di *Ranked Arenas*. Bersaing markah dengan pemain lain untuk naik ke carta teratas!

Peringkat (Rank):
GANGSA ➔ PERAK ➔ EMAS ➔ PLATINUM ➔ BERLIAN ➔ ALPHA

4. Mod Kelas (Classroom)

Sesi interaktif secara langsung (live) yang dikawal sepenuhnya oleh guru.

  • Akses Mudah: Guru mencipta kelas secara automatik. Murid boleh didaftarkan oleh guru atau masuk sendiri menggunakan Kod Kelas / QR Code.
  • Kawalan Penuh: Guru menetapkan jumlah soalan, tahap kesukaran, dan had masa.
  • Aksi Serentak: Apabila guru menekan 'Start', kiraan detik bermula dan semua murid menjawab set soalan yang sama.
  • Live Leaderboard: Guru boleh memantau kedudukan murid secara *live*. Kedudukan akan sentiasa berubah berdasarkan ketepatan jawapan!
  • Rekod Pangkalan Data: Keputusan papan pendahulu (leaderboard) disimpan dalam sistem untuk disemak bila-bila masa.
